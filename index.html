HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The Meta element is a element that contains infomation about the website
  • The title element defines the title for a webpage
  • HTML can include writing, links, pictures, and even sound and video.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is used to style and layout web pages
  • The major structural elements of CSS include style sheets, attribute-value pairs, and binding.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file]: add file as it looks now to your commit (stage)
  • git reset [file]: to unstage a file while retaining changes in working directoty
  • git merge [branch]: merge the specified branchs history into current one
  • git log: shows all commits in the current branchs history

JavaScript

  • Control flow is the order in which a computer executes code in a script.
  • A text interface that allows you to input commands snd view outputs.
  • A variable is a named container that allows us to store data in our code.
  • Conditional statments will start by declaring a variable
  • The two requirments for a JavaScript function is to define the function, then call the function
  • The Console.log are the steps inside of the function.
