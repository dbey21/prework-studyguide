Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The Meta element is a element that contains infomation about the website
  • The title element defines the title for a webpage
  • HTML can include writing, links, pictures, and even sound and video.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • This internal style sheet contains CSS rules for the webpage in the head section of the HTML file.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • The img selector assigns property values to any image element in the markup.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file]: add file as it looks now to your commit (stage)
  • git reset [file]: to unstage a file while retaining changes in working directoty
  • git merge [branch]: merge the specified branchs history into current one
  • git log: shows all commits in the current branchs history
